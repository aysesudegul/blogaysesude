# Business Analysis Blog Portfolio

A clean personal blog and portfolio website built for an aspiring Business Analyst. The site presents short articles, analysis notes, a practical toolkit, and a CV-style summary in a polished, responsive layout.

## What It Includes

- Editorial homepage for business analysis notes and portfolio positioning
- Featured blog topics about requirements analysis, BPMN, SQL, and stakeholder work
- Business analysis toolkit section with practical focus areas
- CV summary timeline and skill tags
- Responsive desktop and mobile design

## Running Locally

```bash
node server.mjs
```

Then open `http://127.0.0.1:4173`.
